# Code of Conduct

## Our Pledge

We are committed to making participation in this project a welcoming experience for everyone, regardless of background or experience level.

## Our Standards

**Do:**
- Be respectful and constructive
- Welcome newcomers and help them get started
- Accept feedback gracefully
- Focus on what's best for the project and community

**Don't:**
- Use hostile or exclusionary language
- Harass or troll other contributors
- Dismiss someone's contribution without explanation
- Share others' private information

## Enforcement

Project maintainers will remove, edit, or reject contributions that violate this code of conduct. Repeated violations may result in a ban from the project.
