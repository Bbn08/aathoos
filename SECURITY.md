# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in aathoos, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Use [GitHub's private vulnerability reporting](https://github.com/aathoos/aathoos/security/advisories/new) to submit your report.

## Supported Versions

As an early-stage project, only the latest version is supported with security updates.

| Version | Supported |
| ------- | --------- |
| latest  | Yes       |
